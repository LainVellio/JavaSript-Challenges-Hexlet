Счастливые числа

Назовем счастливыми числами те, которые в результате ряда преобразований вида "сумма квадратов цифр" превратятся в единицу. Например:

7 => 7² = 49,
49 => 4² + 9² = 16 + 81 = 97,
97 => 9² + 7² = 81 + 49 = 130,
130 => 1² + 3² + 0² = 10,
10 => 1² + 0² = 1.

Вывод: исходное число 7 - счастливое.
